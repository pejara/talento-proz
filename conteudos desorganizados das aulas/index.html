Introdução ao JavaScript

Contextualização do JavaScript

Hoje vamos começar com o JavaScript. Quero contextualizar dizendo que JavaScript é uma linguagem de programação que foi criada por Brendan Eich em 1995. É amplamente utilizada no desenvolvimento front-end para criar interatividade em páginas web. Uma curiosidade interessante é que, apesar do nome semelhante, JavaScript e Java são duas linguagens diferentes. JavaScript foi nomeado assim para capitalizar a popularidade do Java na época, mas suas sintaxes e propósitos são distintos.

Popularidade no Front-End

É uma das linguagens mais utilizadas no desenvolvimento front-end. É essencial para criar experiências dinâmicas e interativas na web.

Declaração de Variáveis

Vamos explorar como declarar variáveis em JavaScript e compará-las com Python:

  • JavaScript:
    • var nome = "João"; - Declara uma variável com escopo global ou local.
    • let idade = 30; - Declara uma variável com escopo de bloco.
    • const cidade = "São Paulo"; - Declara uma variável de valor constante, que não pode ser reatribuído.
  • Python:
    • nome = "João" - Declara uma variável sem especificar o tipo.
    • idade = 30 - Declara uma variável com um valor.
    • cidade = "São Paulo" - Declara uma variável com um valor.

Como Linkar um Arquivo JavaScript no HTML

Para adicionar um arquivo JavaScript ao seu HTML, você precisa usar a tag <script>. Existem duas maneiras principais de fazer isso:

  • Linkar um arquivo JavaScript externo:

    • Você pode linkar um arquivo JavaScript externo usando o atributo src. Por exemplo:

    <script src="script.js" defer></script>

    Isso diz ao navegador para carregar e executar o código JavaScript contido no arquivo script.js. O atributo defer garante que o script seja executado após o HTML ser completamente analisado, melhorando o desempenho da página.

  • Incluir JavaScript diretamente no HTML:

    • Você também pode incluir o código JavaScript diretamente dentro da tag <script>. Por exemplo:

    <script>
    console.log("Olá, Mundo!");
</script>

    Isso executará o código JavaScript diretamente no HTML.
